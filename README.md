# Modern Technology and Society Website

This project is a multi-page HTML website about how modern technology impacts society. The website explores both the positive and negative effects of technology and how it is used in education.

## Webpages Included

- Home Page
- Positive Effects of Technology
- Negative Effects of Technology
- Technology in Education

## Topics Covered

- Communication and social media
- Online learning and education
- Cyberbullying and online safety
- Technology in daily life
- Advantages and disadvantages of technology

## Technologies Used

- HTML
- Basic CSS

## Purpose of the Project

The purpose of this project was to improve web development skills and understand how technology affects society in both positive and negative ways.
